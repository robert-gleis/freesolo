# Team Lifecycle Manager Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `TeamLifecycleManager` under `src/teams/` to create, monitor, and tear down agent teams from `team-plan.json`, emit lifecycle events to the Event Log, and expose `freesolo team start | status | stop` CLI commands.

**Architecture:** Domain logic in `src/teams/` (types, member expansion, monitor helpers, event builders, snapshot store, manager). CLI in `src/commands/team.ts` mirrors `plan.ts` injectable-deps pattern. v1 uses `ScriptedAgentAdapter` via factory injection. `team stop` is snapshot-driven: it reads `team-runtime.json`, emits cancellation events, and updates the snapshot — it does not require a live in-process manager (CLI `start` exits after `create()`).

**Tech Stack:** TypeScript (ES2022, NodeNext), Vitest, Commander; existing `AgentAdapter` (#33), `readTeamPlan` (#34), `EventLog` (#23), `writeState` (#17).

**Note:** Use `PlannerHost` from `src/planner/schemas/team-definition.js`; alias as `AgentHost` in `src/teams/types.ts` for spec alignment.

```ts
import fs from 'node:fs/promises';
import os from 'node:os';
import path from 'node:path';
import { execa } from 'execa';
import { describe, expect, it } from 'vitest';
import { getTeamRuntimePath, readTeamRuntimeSnapshot, writeTeamRuntimeSnapshot } from '../../src/teams/store.js';

async function makeRepo(): Promise<string> {
  const dir = await fs.mkdtemp(path.join(os.tmpdir(), 'freesolo-team-store-'));
  await execa('git', ['init', '--quiet'], { cwd: dir });
  return dir;
}

describe('team runtime store', () => {
  it('round-trips snapshot via git freesolo path', async () => {
    const worktreePath = await makeRepo();
    const snapshot = { issueNumber: 41, phase: 'running' as const, members: [] };
    await writeTeamRuntimeSnapshot(worktreePath, snapshot);
    const runtimePath = await getTeamRuntimePath(worktreePath);
    expect(await readTeamRuntimeSnapshot(worktreePath)).toEqual(snapshot);
    expect(runtimePath).toContain('team-runtime.json');
  });
});
```
